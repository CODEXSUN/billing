# CODEXSUN Billing

Billing and financial document application for CODEXSUN.

This repository owns its backend modules, frontend modules, migrations, workers, permissions, navigation contributions, and tests. It integrates with the generic Platform through the public CodexsunApp contract.

Read `assist/AGENT-GUIDE.md` before changing Billing. Repository ownership, migration/seed
order, environment behavior, versioning, and Git workflow are documented under `assist/`.
